## Notas de Uso para Generación de Imágenes

- **Modelos recomendados para generar:** Midjourney v7, Stable Diffusion XL + ControlNet, DALL-E 3, Adobe Firefly 3, Ideogram 2.0
- **Parámetros adicionales sugeridos en Midjourney:** `--ar 16:9` para banners, `--ar 4:5` para Instagram, `--ar 3:2` para editorial web
- **Estilo de referencia artística:** Para todos los prompts se puede agregar al final: `--style raw` en Midjourney para máximo realismo
- **Consistencia de madera:** En todas las generaciones, la madera debe mostrar veta de pino visible, color marrón cálido (hex aproximado #8B6914 para pino barnizado), bordes ligeramente redondeados
- **Revisión post-generación:** Verificar que la madera no sea de color gris (pino envejecido) ni demasiado oscuro (roble) — debe ser el tono rubio-dorado del pino con barniz color natural
